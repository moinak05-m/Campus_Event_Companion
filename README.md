# Campus Event Companion App 🎓

A smart platform that helps students discover, register, and manage campus events from one centralized dashboard.

## 🚀 Features

- Event discovery
- Student dashboard
- Event registration
- Admin event management
- Notifications & reminders
- Responsive modern UI
- Authentication system
- MongoDB database integration

## 🛠️ Tech Stack

### Frontend
- React
- Vite
- CSS / Tailwind

### Backend
- Node.js
- Express.js
- MongoDB Atlas

## 📂 Project Structure

```bash
frontend/
backend/
```

## ⚙️ Installation

### Clone Repository

```bash
git clone <your-repo-link>
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## 🌐 Environment Variables

Create `.env` inside backend:

```env
PORT=5000
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret
```
